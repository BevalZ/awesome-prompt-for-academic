### 学術研究の問いジェネレーター

**Tags:** `研究方法論` | `研究計画`

**Description:** 学術論文やプロジェクトのための焦点を絞った研究の問いを生成します。このプロンプトは、文献レビューや実験計画を導く、明確で簡潔、かつ研究可能な問いの策定に役立ちます。

**Prompt:**
```
経験豊富な学術研究者として、[トピック/分野]に関する明確で焦点を絞った研究の問いを策定するのを手伝ってください。

背景: [あなたの研究背景、分野、目的の簡単な説明]

要件:
- 問いは具体的で研究可能であること
- 分野における重要なギャップや問題に取り組むこと
- 一般的な学術的制約の中で実現可能であること
- 実行可能な研究の方向性につながること

提供してください:
1. 主要な研究の問い
2. 主な問いをサポートする2〜3の副次的な問い
3. この問いが重要である理由の簡単な正当化
4. それに取り組むための提案された方法論やアプローチ
5. 分野への潜在的な影響や貢献

構造化された研究提案の概要として回答をフォーマットしてください。
```

### 文献レビューシンセサイザー

**Tags:** `文献レビュー` | `知識の統合`

**Description:** 複数の情報源を統合して、包括的な文献レビューを作成します。このプロンプトは、学術文献を効果的に整理、分析、統合するのに役立ちます。

**Prompt:**
```
[トピック]に関する文献レビューを作成し、[数]の主要な情報源からの知見を統合する必要があります。

含めるべき情報源:
[簡単な引用とともに情報源をリストアップ]

研究の焦点: [あなたが取り組んでいる特定の側面や問い]

提供してください:
1. 研究の現状の概要
2. 情報源全体にわたる主要なテーマとパターンの特定
3. 方法論と知見の批判的分析
4. 文献におけるギャップや矛盾
5. 知見を首尾一貫した物語に統合
6. 将来の研究の方向性のための提言
7. [引用スタイル]形式での適切な学術引用

アイデア間の明確な見出しとスムーズな移行でレビューを構成してください。
```

### 学術的執筆の明確さ向上

**Tags:** `学術的執筆` | `学術的執筆`

**Description:** 学術的なトーンと厳密さを維持しながら、学術的執筆の明確さと読みやすさを向上させます。

**Prompt:**
```
学術的な厳密さを維持しながら、この学術テキストの明確さと読みやすさを向上させる手伝いをしてください:

[ここにあなたのテキストを挿入]

焦点を当てる点:
- 文の構造と流れ
- 言葉の選択と正確さ
- 論理的な構成
- アイデア間の移行
- 冗長性の排除
- 専門用語が適切に説明されていることの確認

提供してください:
1. 改善点を強調表示した改訂版テキスト
2. 行われた主要な変更の簡単な説明
3. さらなる改善のための提案
4. 今後のライティングでこの明確さを維持するためのヒント

元の意味と学術的なトーンを全体を通して維持してください。
```

```
「」に関する研究のためのトップ5の学術ジャーナルを特定してください。それぞれについて、その範囲と典型的な方法論的焦点を説明してください。

```

### 学術論文の要約と分析

**Tags:** `読解` | `論文分析`

**Description:** 学術論文を300語で要約し、その研究の問い、方法論、主要な発見、および主な結論を抽出します。論文の核心部分を迅速に理解するのに役立ちます。

**Prompt:**
```
次の論文を300語で要約してください。その研究の問い、方法論、主要な発見、および主な結論を抽出してください。
```

### アブストラクトの批判的査読者

```
批判的な査読者として行動してください。次のアブストラクトを読み、研究のデザインや主張における3つの潜在的な弱点を特定してください。
```

### 理論的枠組みの比較分析

**Prompt:**
```
次の2つの論文で使用されている理論的枠組みを比較対照してください。比較を表形式で提示してください。
```

### 方法論の妥当性分析

**Tags:** `研究方法論` | `方法論分析`

**Description:** 研究論文の方法論を分析し、研究の問いに対するその妥当性を評価し、その主要な長所と限界を特定します。研究の批判的評価に不可欠です。

**Prompt:**
```
添付の論文の方法論を分析してください。研究の問いに対して適切ですか？その主要な長所と限界は何ですか？
```
