### 学术研究问题生成器

**Tags:** `研究方法论` | `研究规划`

**Description:** 为学术论文和项目生成重点突出的研究问题。此提示有助于形成清晰、简洁且可研究的问题，从而指导文献综述和实验设计。

**Prompt:**
```
作为一名经验丰富的学术研究者，请帮助我为[主题/领域]制定一个清晰且重点突出的研究问题。

背景：[简要描述您的研究背景、领域和目标]

要求：
- 问题应具体且可研究
- 问题应针对该领域的重大空白或问题
- 问题在典型的学术限制内应是可行的
- 问题应能引导出可行的研究方向

请提供：
1. 一个主要研究问题
2. 2-3个支持主要问题的次要问题
3. 简要说明此问题为何重要
4. 解决此问题的建议方法或途径
5. 对该领域的潜在影响或贡献

请以结构化的研究计划大纲格式作出回应。
```

### 文献综述综合器

**Tags:** `文献综述` | `知识综合`

**Description:** 通过综合多个来源创建全面的文献综述。此提示有助于有效组织、分析和综合学术文献。

**Prompt:**
```
我需要就[主题]创建一篇文献综述，综合来自[数量]个关键来源的发现。

要包括的来源：
[列出您的来源及其简要引文]

研究重点：[您正在处理的特定方面或问题]

请提供：
1. 研究现状概述
2. 跨来源的关键主题和模式识别
3. 方法论和发现的批判性分析
4. 文献中的空白或不一致之处
5. 将发现综合成连贯的叙述
6. 对未来研究方向的建议
7. [引文样式]格式的正确学术引文

请以清晰的标题和流畅的思路过渡来构建综述。
```

### 学术写作清晰度改进器

**Tags:** `学术写作` | `学术写作`

**Description:** 在保持学术语气和严谨性的同时，提高学术写作的清晰度和可读性。

**Prompt:**
```
请帮助提高此学术文本的清晰度和可读性，同时保持其学术严谨性：

[在此处插入您的文本]

重点关注：
- 句子结构和流畅性
- 词语选择和精确性
- 逻辑组织
- 思路之间的过渡
- 消除冗余
- 确保专业术语得到适当解释

请提供：
1. 突出显示改进之处的修订文本
2. 对所做关键更改的简要说明
3. 对进一步改进的建议
4. 在未来写作中保持这种清晰度的技巧

在整个过程中保持原文的意义和学术语气。
```

### 文献检索综合关键词及同义词生成器

**Tags:** `文献综述` | `关键词生成`

**Description:** 为特定主题的文献检索生成全面的关键词、同义词、广义词和狭义词列表。这有助于确保制定彻底有效的检索策略。

**Prompt:**
```
为关于“”主题的文献检索生成一个全面的关键词和同义词列表。包括广义词和狭义词。
```

**Prompt:**

### 方法论恰当性分析

**Tags:** `研究方法论` | `方法论分析`

**Description:** 分析一篇研究论文的方法论，评估其对于研究问题是否恰当，并指出其主要优点和局限性。对于批判性地评估研究至关重要。

**Prompt:**
```
分析所附论文的方法论。它是否适合该研究问题？其主要优点和局限性是什么？
```
